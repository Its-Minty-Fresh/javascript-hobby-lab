Click Counter

Click Counter

Requirements

  1. First, create an HTML page with two buttons.
  2. The first button, when clicked, adds one to a variable that keeps track of the number of clicks.
  3. The second button, when clicked, uses an alert() to let the user know how many times the first button was clicked.
  4. The message should say "You clicked x times" where x is the number of times the first button was clicked.
